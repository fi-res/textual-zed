# textual-zed

[Textual](textual.textualize.io) CSS extension for syntax highlighting for Zed.

Tree sitter: [tree-sitter-tcss](https://github.com/fi-res/tree-sitter-tcss)

## Features
 - Syntax highlighting
 - Auto indenation and brackets completion
 - Folds
 - Outline

## Installation

1. Clone project
```bash
git clone https://github.com/fi-res/textual-zed
```
2. In Zed, click `Extensions` -> `Install dev extension`
3. Select project folder
4. Open any .tcss file
